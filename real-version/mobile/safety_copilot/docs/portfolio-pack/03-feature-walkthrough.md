# 03 - Feature Walkthrough

## 1) Authentication
- Register with name/phone/password
- Login and token session

## 2) Trusted Circle
- Create circle (e.g., Family)
- Add members by phone

## 3) Safe Trip Lifecycle
- Start trip with destination
- Send periodic live location and battery status
- Auto-detect arrival and notify circle
- End trip manually when needed

## 4) Safety Events
- SOS: visible emergency signal
- Silent SOS: discreet emergency signal
- Auto alerts: low battery, offline risk, route anomaly, delay

## 5) Alerts Feed
- View recent alerts
- Acknowledge alert to close response loop
