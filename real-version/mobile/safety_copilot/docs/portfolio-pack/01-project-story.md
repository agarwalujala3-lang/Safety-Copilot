# 01 - Project Story

## Problem
Daily travel safety is still reactive. Families usually know something is wrong only after a delay.

## Solution
Safety Copilot is a trusted-circle safety app that combines:
- Live trip tracking
- Automatic safety alerting
- SOS and Silent SOS workflows
- Arrival intelligence

## Why it matters
It is useful both on normal days (routine visibility) and risky situations (fast emergency escalation).

## Your contribution
You designed and shipped:
- Full-stack architecture (mobile + backend + cloud)
- Safety event model (arrival/offline/low battery/SOS)
- Release-ready Android builds (AAB + APK)
- Compliance artifacts (privacy policy + data safety mapping)
