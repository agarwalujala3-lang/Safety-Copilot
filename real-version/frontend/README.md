# Safety Copilot Frontend

Animated operator dashboard for the safety backend.

## Stack

1. React + Vite
2. Framer Motion (UI motion)
3. Three.js via `@react-three/fiber` + `@react-three/drei` (3D visual core)

## Start

```bash
npm install
npm run dev
```

Open `http://localhost:5173`

## Features wired to API

1. Register/login
2. Create trusted circle
3. Add member
4. Start active trip
5. Send live location pings
6. Trigger SOS/silent SOS
7. See live alerts + location stream
